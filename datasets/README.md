# CircuitBench Datasets

This directory contains all benchmark datasets used throughout CircuitBench.

## Categories

- Analog
- Digital
- Passive
- Power Electronics
- RF
- Mixed Signal
- Synthetic
- External Validation

## Directory Layout

raw/
interim/
processed/
features/
validation/
metadata/
statistics/
reports/
schema/
scripts/

Each dataset should include:

- metadata
- license
- citation
- checksum
- validation report
- preprocessing documentation
- feature description
