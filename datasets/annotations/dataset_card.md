# Annotations Dataset

## Overview

This directory contains annotation resources used throughout CircuitBench.

## Purpose

The annotation datasets provide standardized labels and metadata describing electrical circuits, simulation outputs, benchmark tasks, and evaluation targets.

## Contents

- Circuit annotations
- Component annotations
- Signal annotations
- Task annotations
- Metadata labels

## Intended Users

- Machine Learning researchers
- Circuit designers
- Dataset contributors

## Inputs

Raw circuit data.

## Outputs

Structured annotations.

## Quality Assurance

Annotations are reviewed for consistency and schema compliance.
