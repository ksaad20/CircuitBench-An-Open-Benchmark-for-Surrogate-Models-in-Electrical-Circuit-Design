# Circuit-Bench Benchmark Specification

## 1. Purpose

Circuit-Bench is a standardized benchmark for evaluating artificial intelligence methods in electrical circuit design. Its objective is to provide a common evaluation platform that enables fair, reproducible and comparable assessment of machine learning algorithms across multiple electrical engineering tasks.

## 2. Scope

Circuit-Bench evaluates AI algorithms for

• Analog circuits

• Digital circuits

• Power electronics

• Passive networks

• RF circuits

• Mixed-signal circuits

## 3. Benchmark Tasks

Task 1
Circuit performance prediction

Task 2
Component parameter estimation

Task 3
Circuit optimisation

Task 4
Fault diagnosis

Task 5
Sensitivity analysis

Task 6
Surrogate modelling
