# Visualization

CircuitBench provides visualization utilities for simulation and machine learning results.

Supported plots

- Bode plots
- Waveforms
- Nyquist plots
- Pole-zero plots
- Learning curves
- Confusion matrices
- ROC curves
- Precision-Recall curves
- Feature Importance
- SHAP values

Visualization outputs are publication-ready and can be exported as PNG, SVG, and PDF.
